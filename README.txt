Happy New Year
https://github.com/itcrowd72/happy_new_year
==============================================

DESCRIPTION
-----------
A module with garland and snow decoration that creates longlasting atmosphere of New Year for you and the users of your website.

INSTALLATION AND CONFIGURATION
------------------------------
1) Install the module as usual.
2) Enable the module.
3) Go to admin/config/media/happy_new_year and enable garland and snow.
